# Dehenzify

Dehenzifies code!

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dehenzify'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dehenzify

## Usage example

    $ bundle exec dehenzify app/lib/stats
